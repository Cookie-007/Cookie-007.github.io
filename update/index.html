# 2025/3

  • 23号大脑突发奇想，和小脑一拍即合，二者说干就干，随之咨询请教了zmal，用Hexo结合Github搭建了静态渲染网页，那时候什么都不会，一个小东西也要问ai问zmal找各种文章，一开始是想去b站学的，但是记得zmal好像在这方面已经炉火纯青了，所以干脆请教他了，后面我就后悔了，因为我从0起手，而zmal的搭建方法和b站的又略有差异，所以我前期遇到问题几乎都是问ai和zmal，而ai也是真只能解决我小部分问题，我也不好意思天天问zmal，git的文档对新手来说也是有点晦涩...他也算是给我一个烂摊子了，经过我各种看文档、找知乎，问zmal等等等等，终于是把网站建起来了，尽管此时几乎只有一个空壳。
